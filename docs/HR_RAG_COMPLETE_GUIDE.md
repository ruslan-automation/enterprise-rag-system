# HR RAG Bot - Полное руководство

## 📋 Что построено

**HR RAG Assistant** — AI-бот для Telegram, который отвечает на вопросы сотрудников по внутренним политикам компании "Гравитация".

### Технологии:
- **n8n** — оркестрация воркфлоу
- **Pinecone** — векторная база знаний
- **Cohere** — создание эмбеддингов
- **OpenRouter (DeepSeek)** — генерация ответов
- **Telegram** — интерфейс бота
- **Google Sheets** — логирование запросов

**1. Хук (0:00-0:03)**
*   **Визуал:** Твое лицо крупно или текст на черном фоне.
*   **Текст/Аудио:** "HR отдел тратит 5 часов в день на ЭТИ вопросы..." или "Я заменил HR-менеджера нейросетью?"

**2. Демонстрация (0:03-0:20)**
*   *Запись экрана смартфона.*

*   **Вопрос 3 (Юмор):** "Опоздал на дейли, что будет?" → *Ответ про пиццу.*
*   *Фишка:* Демонстрация того, что бот понимает живую речь, а не только сухой текст.

**3. "Под капотом" / Экспертность (0:20-0:25)**
*   *Быстрый зум на схему воркфлоу в n8n.*
*   **Текст:** "Голос → Whisper → RAG → Ответ."
*   **Озвучка:** "Распознает голос, ищет в базе, отвечает."

**4. CTA (Call to Action) (0:25-0:30)**
*   **Визуал:** Ты показываешь пальцем вниз или на текст.
*   **Текст:** "Забирай схему бота."
*   **Озвучка:** "Пиши слово 'RAG', скину полную инструкцию и JSON схемы."

### 💡 Советы по съемке:
*   **Формат:** 9:16 (Вертикально).

## 🏗️ Архитектура воркфлоу

### Legal_RAG_Chat (Основной бот)
```
Telegram Trigger
    ↓
Extract Question (Code)
    ↓
Embed Question (Cohere API)
    ↓
Pinecone Query (Поиск в базе)
    ↓
Format Context (Code)
    ↓
Legal AI Response (OpenRouter)
    ↓
Send Telegram Response
    ↓
Log to Sheets (Аналитика)
```

### Legal_RAG_Ingestion (Загрузка данных)
```
Manual Trigger
    ↓
Read HR Policies (Google Sheets)
    ↓
Loop over rows:
    Create Embeddings (Cohere)
        ↓
    Upsert to Pinecone
```
